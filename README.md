# 🌍 Problem Statement

Urban traffic collapses caused by cricket matches, political rallies, religious processions, concerts, accidents, and public gatherings are often predictable. However, existing traffic systems react only after congestion has already occurred.

This leads to:

- Severe congestion
- Increased travel times
- Delayed emergency response
- Higher fuel consumption and emissions
- Economic losses

UrbanFlow AI addresses this challenge by enabling predictive traffic intelligence.

---

# 💡 Solution

UrbanFlow AI is an AI-powered traffic intelligence platform designed to anticipate congestion before it happens.

By combining event intelligence, analytics, forecasting, and AI assistance, UrbanFlow AI helps cities move from reactive traffic management to proactive decision-making.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/siddharth182005-dotcom/UrbanFlow_AI_Hackathon_Submission.git
```

Move to project directory:

```bash
cd UrbanFlow_AI_Hackathon_Submission
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
